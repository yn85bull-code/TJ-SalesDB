# TJ-SalesOS / 超新世界リンクOS 参照ルール

このメモは今後のTJ-SalesOS、GAS、HTML、WEBダッシュボード開発の基準とする。

## 基本構成

今後のデータ導線は以下を原則とする。

```text
各種CSV
↓
各種集計シート
↓
超新世界リンクOS
↓
WEBダッシュボード
```

WEB側は原則として `超新世界リンクOS` を参照する。
足りない項目がある場合のみ元シートを見る。

営業部ごとにHTMLやGASを増やさない。
営業部プルダウンで `参照設定` シートを読み、対象営業部のスプレッドシートID・シート名を切り替える。

## 開発優先順位

1. 超新世界リンクOSを読む
2. 足りない場合のみ元シートを見る
3. 可能な限り超新世界リンクOSを基準にする
4. 参照設定シートで営業部ごとの参照先を切り替える
5. WEB画面は1つに統一する
